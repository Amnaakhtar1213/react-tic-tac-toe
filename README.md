# TIC-TAC-TOE

A simple tic-tac-toe game built with **REACT**. Player take turns placing X and O on a 3×3 grid. The game automatically detect winner, draw and allow resetting the game.

---

## Features

1. Nine clickable aquares buttons
2. Display a winner, when player wins
3. Reset button to restart game
4. Centered layout with style button 

---

## Technolog

- **React**   
- **css**
