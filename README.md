# TIC-TAC-TOE

A simple built tic-tac-toe game with **REACT**.9 squares block, reset button and reault displaying div.

---

## Features

1. changing turn (X , O)
2. reset button to set the game back to new
3. A result displaying div to announce winner 

---

## Technolog

- **React**   
- **css**
